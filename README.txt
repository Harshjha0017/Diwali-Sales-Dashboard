🪔 Diwali Sales Dashboard Project

This project showcases a professional sales dashboard created with Google Sheets, which analyzes sales data from a Diwali campaign.

📊 Key Features
- KPIs: Total Revenue, Orders, AOV, Customer Count
- Demographic Analysis (Gender, Age, Occupation)
- Regional Insights (Top States, Revenue Split)
- Product & Category Performance
- Strategic Recommendations and Action Panel

📁 Files Included
- `Dashboard (Diwali Sales).jpg`: The complete dashboard
- `Insights.md`: Analytical observations
- `Charts_Explained.md`: Chart breakdown and purpose

✅ Skills Demonstrated
- Data Cleaning
- KPI Design
- Visual Storytelling
- Business Insight Extraction
